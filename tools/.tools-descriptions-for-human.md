# form-auto-submit.py

**Role:** Crawl các form trên danh sách endpoint, tự điền và submit qua ZAP proxy để ZAP ghi lại request vào history — cung cấp coverage cho active scan ở bước sau.

**Chạy thủ công trước khi gọi use-ZAP.py.**

## Usage

```bash
py tools/form-auto-submit.py -i <endpoint_file> -s <session_file> [options]

# Example:
py tools/form-auto-submit.py -i target_info\site-endpoints.txt -s target_info\session.txt
```

## Arguments

| Arg | Default | Mô tả |
|-----|---------|-------|
| `-i / --input-file` | `katana.filtered.txt` | File chứa danh sách endpoint (1 URL/path mỗi dòng) |
| `-s / --session` | *(none)* | Path đến session file (`key=val; key=val`) |
| `-o / --output-file` | `katana_filtered_2.txt` | Output các endpoint đã submit (cho debug) |
| `--agent-output` | *(same as -o)* | Output clean form HTML + submit URL (cho agent đọc context) |
| `-b / --base-url` | `http://192.168.153.200/DVWA` | Base URL của target |
| `-p / --proxy` | `http://192.168.153.130:8080` | ZAP proxy URL |
| `--auth-user` | `admin` | Username nếu cần auto login |
| `--auth-pass` | `password` | Password nếu cần auto login |
| `--no-auth` | false | Bỏ qua auto login |
| `--cookie` | *(none)* | Raw cookie string thay thế (`"a=b; c=d"`) |

## Input

**`endpoint_file`** — 1 URL hoặc path mỗi dòng. Path tương đối sẽ được ghép với `base_url`.

```
/vulnerabilities/xss_s/
/vulnerabilities/sqli/
http://192.168.153.200/DVWA/login.php
```

**`session_file`** — Cookie string dạng `key=val; key=val`:

```
PHPSESSID=1lsg7f8o7fjg75agf20t0dp7gi; security=low
```

## Output

**`agent-output`** — Mỗi form block phân cách bởi dòng trống:

```
<form method="post" name="guestform">
  <input name="txtName" type="text"/>
  <textarea name="mtxMessage"></textarea>
  <input name="btnSign" type="submit" value="Sign Guestbook"/>
</form>
_
http://192.168.153.200/DVWA/vulnerabilities/xss_s/
```

*(Đây chính là `target_info/site-forms.txt`)*

## Behavior Notes

- **Tự động skip** các form/URL có từ khoá nguy hiểm: `logout`, `reset`, `delete`, `drop`, `clear`, v.v.
- **Tự động login** nếu detect login page và không có session.
- **Skip file input** (`type="file"`) — không upload file.
- **Chỉ submit 1 submit button** đầu tiên mỗi form (tránh double-submit).
