```bash
py tools/form-auto-submit.py -i katana.filtered.txt --sid 1lsg7f8o7fjg75agf20t0dp7gi
```

use-ZAP.py
Gọi ZAP pentest các URL đã lọc và lấy kết quả raw:

```bash
py tools/use-ZAP.py -i <endpoint_file> -b <base_url> --sid <PHPSESSID> --fast-scan -o <output_file> --proxy-host <proxy_host> --proxy-port <proxy_port>

# Example:
py tools/use-ZAP.py -i katana.filtered.txt -b http://192.168.153.200/DVWA/ --sid "1lsg7f8o7fjg75agf20t0dp7gi" --fast-scan -o zap_results.json --proxy-host 192.168.153.130 --proxy-port 8080
```

compress-results.py
Lọc tất cả kết quả từ tool trước đó, loại bỏ các kết quả THỰC SỰ VÔ NGHĨA:

```bash
py tools/compress-results.py -i <input_file> -o <output_file> --pretty

# Example:
py tools/compress-results.py -i zap_results.json -o zap_compressed.json --pretty
```

zap_phase2_reducer.py
Lọc raw response:

```bash
py .\tools\zap_phase2_reducer.py -i zap_compressed.json -o zap_reduced.json
```
